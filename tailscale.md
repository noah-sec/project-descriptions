<h1 align=center>Zero-Trust Fabric for Identity-Driven, Application-Aware, <br>& Policy-Defined Resource Access</h1>

<!-- INTRODUCTION -->
<p align=center><i>Designed and deployed a software-defined mesh overlay network on top of the existing physical network plane. Used Tailscale to control routing between devices based on continuous identity authentication, connecting device posture, and application-aware policies. This <br>produced a flat network across multiple LANs with strict access controls between users, devices, and services.</i></p>
